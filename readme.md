BCURRAN3's Chocolatey Packages (2025+)

This repo contains:
* My 3rd party Chocolatey utilities
* Packages I use that are not available on the Chocolatey Community Repository or not currently maintained on the Chocolatey Community Repository

These packages are served via my publically available private Chocolatey Repository found at https://nuts.bcurran3.org. You can view the feed at https://nuts.bcurran3.org/packages. To install a package, follow the instructions on the feed's package page. To add the repository to your sources for easy searching, installing, and updating type 'choco sources add -n=BCURRAN3 -s=https://nuts.bcurran3.org/nuget/chocolatey/ --priority=1' into a command prompt. (Priority 0 is highest and you should reserve that for the CCR.)

I hope these packages may serve you well as they do me.
